# depo下载说明
请先在终端执行此命令

`git clone https://github.com/6w9bilibili/depo.git`

`pkg install bash`

接着找到depo文件夹

`cd depo`

如果你是Android用户，请先在termux中执行以下命令

`termux-setup-storage`
  
`pkg install git`

对了，如果您想要快捷方式，可参考下面

Linux路径-

`sudo cp depo.sh /usr/local/bin/depo
sudo chmod +x /usr/local/bin/depo

Android路径

`~/bin/depo'

`chmod +x ~/bin/depo`

`echo 'export PATH=$HOME/bin:$PATH' >> ~/.bashrc
source ~/.bashrc`
